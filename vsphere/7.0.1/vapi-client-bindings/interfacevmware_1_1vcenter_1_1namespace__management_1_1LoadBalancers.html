Detailed Description

LoadBalancers represent the user provisioned load balancers.

The load balancers provided may be used to front the API servers in both, vSphere Namespaces Cluster and Tanzu Kubernetes Grid clusters. Note: The lifecycle of these load balancers is not managed by vSphere. This interface was added in vSphere API 7.0.1.0.
