Detailed Description

The Subscriptions.UpdateSpecVcenter class defines information about the vCenter Server instance where the subscribed library associated with the subscription exists.

The Subscriptions.UpdateSpecVcenter class is only applicable to subscribed library which exists on remote vCenter Server instance. This class was added in vSphere API 6.7.2.
