Detailed Description

The CustomizationSpecs.FilterSpec class contains fields used to filter the results when listing guest customization specifications (see CustomizationSpecs.List ).

If multiple fields are specified, only guest customization specifications matching all of the fields match the filter. This class was added in vSphere API 6.7.1.
