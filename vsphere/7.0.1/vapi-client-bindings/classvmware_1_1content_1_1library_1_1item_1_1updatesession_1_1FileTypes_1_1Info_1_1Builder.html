Builder vmware::content::library::item::updatesession::FileTypes::Info::Builder::SetKeepInStorage ( bool?  keepInStorage) [inline]

Whether or not the file will be kept in storage upon update session completion.

The flag is true for most files, and false for metadata files such as manifest and certificate file of update session with library item type OVF. Any file with File.Info.keepInStorage set to false will not show up in the list of files returned from vmware.content.library.item.File.List upon update session completion. This attribute was added in vSphere API 6.8.
