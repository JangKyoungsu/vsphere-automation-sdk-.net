Builder vmware::vcenter::deployment::MigrateTypes::MigrateSpec::Builder::SetHistory ( vmware.vcenter.deployment.HistoryMigrationSpec  history) [inline]

Determines how vCenter history will be migrated during the migration process.

vCenter history consists of:

  • Statistics
  • Events
  • Tasks

By default only core data will be migrated. Use this spec to define which part of vCenter history data will be migrated and when. This attribute was added in vSphere API 7.0.0.0.
