Detailed Description

The Clusters.WorkloadNetworksEnableSpec contains the specification required to configure workload networks for a vSphere Namespaces Cluster during Enable operation.

These workload networks will be used as backing network for Tanzu Kubernetes Cluster VMs and Kubernetes control plane VMs. This class was added in vSphere API 7.0.1.0.
