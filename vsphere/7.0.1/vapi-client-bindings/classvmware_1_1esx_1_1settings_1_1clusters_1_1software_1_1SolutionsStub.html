Deletes the given solution from the desired software specification.

The deletion will be validated along with the entire software specification before it is committed as new desired spec. The result of this operation can be queried by calling the cis/tasks/{task-id} where the task-id is the response of this operation.

Task method overload. Result of the invocation is a task identifier which is going to be returned by this method.
