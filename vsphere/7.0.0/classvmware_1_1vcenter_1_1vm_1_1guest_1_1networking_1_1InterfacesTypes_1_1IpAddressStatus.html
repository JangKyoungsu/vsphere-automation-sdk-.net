Detailed Description

The Interfaces.IpAddressStatus enumerated type defines the present status of an address on an interface.

See RFC 4293 IpAddressStatusTC. This enumeration was added in vSphere API 7.0.0.

Special enumeration constant which is used to represent constants that do not exist in the current enum binding.
