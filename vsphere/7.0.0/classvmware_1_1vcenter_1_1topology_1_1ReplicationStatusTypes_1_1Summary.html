Detailed Description

The ReplicationStatus.Summary class contains replication information of partner vCenter or Platform Services Controller node of type VCSA_EMBEDDED/PSC_EXTERNAL (see Nodes.Info.type ).

This class was added in vSphere API 6.7.2.
