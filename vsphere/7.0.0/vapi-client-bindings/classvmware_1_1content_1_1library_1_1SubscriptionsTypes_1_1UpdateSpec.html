vmware.content.library.SubscriptionsTypes.UpdateSpecPlacement vmware::content::library::SubscriptionsTypes::UpdateSpec::GetSubscribedLibraryPlacement ( ) [inline]

Placement specification for the virtual machine template items of the subscribed library.

Updating this information will only affect new or updated items, existing items will not be moved. The entire placement configuration of the subscribed library will replaced by the new specification. This attribute was added in vSphere API 6.7.2.
