Identifier of the Content Library which holds the VM Images for vSphere Kubernetes Service.

This Content Library should be subscribed to VMware's hosted vSphere Kubernetes Service Repository. When clients pass a value of this class as a parameter, the field must be an identifier for the resource type: com.vmware.content.Library . When methods return a value of this class as a result, the field will be an identifier for the resource type: com.vmware.content.Library .
