VCHA Cluster network configuration of the node.

All cluster communication (state replication, heartbeat, cluster messages) happens over this network. The active node's management vCenter server credentials are not required to populate Cluster.WitnessInfo.haIp . This attribute was added in vSphere API 6.7.1.
