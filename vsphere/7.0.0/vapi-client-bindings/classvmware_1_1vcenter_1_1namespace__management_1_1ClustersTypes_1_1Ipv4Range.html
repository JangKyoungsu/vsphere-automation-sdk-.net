Detailed Description

The Clusters.Ipv4Range contains specification to configure multiple interfaces in IPv4.

The range of IPv4 addresses is derived by incrementing the startingAddress to the specified addressCount. To use the object for a single IPv4 address specification, set addressCount to 1.
