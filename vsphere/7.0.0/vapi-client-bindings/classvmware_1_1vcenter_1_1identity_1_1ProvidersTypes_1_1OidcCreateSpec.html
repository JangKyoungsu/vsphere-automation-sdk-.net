The map used to transform an OAuth2 claim to a corresponding claim that vCenter Server understands.

Currently only the key "perms" is supported. The key "perms" is used for mapping the "perms" claim of incoming JWT. The value is another map with an external group as the key and a vCenter Server group as value. This attribute was added in vSphere API 7.0.0.
