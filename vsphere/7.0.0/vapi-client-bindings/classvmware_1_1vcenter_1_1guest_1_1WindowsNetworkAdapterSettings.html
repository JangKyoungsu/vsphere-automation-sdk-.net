global::System.Collections.Generic.List<string> vmware::vcenter::guest::WindowsNetworkAdapterSettings::GetDnsServers ( ) [inline]

A list of server IP addresses to use for DNS lookup in a Windows guest operating system.

These servers should be specified in the order of preference. If this list is not empty, and if a DHCP IP address is used, then these settings override the DHCP settings.. This attribute was added in vSphere API 7.0.0.

global::System.Collections.Generic.List<string> vmware::vcenter::guest::WindowsNetworkAdapterSettings::GetWinsServers ( ) [inline]

List of WINS Servers to set for the Windows guest operating system.

A Maximum of two IP addresses can be specified in this list. The first IP address will be set as the primary WINS server. The second IP address will be set as the secondary WINS server. This attribute was added in vSphere API 7.0.0.
